ESP32 是什麼？

ESP32 是由 Espressif 開發的微控制器， 內建 WiFi 與 Bluetooth， 廣泛應用於 IoT、智慧家庭、 AIoT、感測器控制等領域。

ESP32 DevKit V1

最常見的 ESP32 開發板， 適合初學者與教學使用。

  • 雙核心 CPU
  • WiFi + Bluetooth
  • 30~38 GPIO

ESP32-WROOM-32

常見模組版本， 許多開發板都使用此晶片。

  • 內建 PCB 天線
  • 低功耗
  • IoT 應用廣泛

ESP32-S3

新世代 ESP32， 支援 AI 與 USB OTG。

  • AI 指令加速
  • USB 支援
  • 更多 GPIO

ESP32-C3

RISC-V 架構， 低功耗且價格便宜。

  • RISC-V CPU
  • 超低功耗
  • 適合 IoT 裝置
