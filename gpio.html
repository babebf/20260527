ESP32 GPIO Pinout

常用 GPIO 腳位

GPIO 功能 用途
GPIO 21 I2C SDA OLED、感測器
GPIO 22 I2C SCL OLED、感測器
GPIO 23 SPI MOSI SPI 傳輸
GPIO 18 SPI SCK SPI 時脈
GPIO 2 內建 LED LED 控制
GPIO 34 ADC Input 類比輸入

GPIO 注意事項

  • GPIO34~39 只能輸入
  • GPIO6~11 為 Flash 使用
  • GPIO0 影響 Boot 模式
  • GPIO2 常作為內建 LED
