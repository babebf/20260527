LED 模組介紹

LED 是最基礎的電子元件， 可控制燈光亮滅， 常用於學習 GPIO 輸出。

材料準備

  • ESP32 開發板
  • LED
  • 220Ω 電阻
  • 麵包板
  • 杜邦線

GPIO 腳位介紹

腳位 功能
GPIO23 LED 訊號輸出
GND 接地

接線方式

  • LED 長腳 → GPIO23
  • LED 短腳 → 電阻 → GND

接線範例圖

Arduino 程式碼



int led = 23;

void setup() {

    pinMode(led, OUTPUT);

}

void loop() {

    digitalWrite(led, HIGH);

    delay(1000);

    digitalWrite(led, LOW);

    delay(1000);

}

程式說明

digitalWrite(HIGH) 會讓 LED 亮起。 digitalWrite(LOW) 會讓 LED 熄滅。 delay(1000) 表示等待 1 秒。
