LED 模組教學

LED 模組介紹

LED 是最基礎的電子元件， 可以控制亮滅與閃爍。

材料準備

  • ESP32 開發板
  • LED
  • 220Ω 電阻
  • 麵包板

接線方式

  • LED 正極 → GPIO23
  • LED 負極 → GND

Arduino 程式


int led = 23;

void setup() {

    pinMode(led, OUTPUT);

}

void loop() {

    digitalWrite(led, HIGH);
    delay(1000);

    digitalWrite(led, LOW);
    delay(1000);

}
