ESP32 功能介紹

首頁 ESP32版本 GPIO腳位

WiFi 無線網路

ESP32 內建 WiFi， 可建立 WebServer、 IoT 裝置、 雲端控制系統。

Bluetooth 藍牙

支援 BLE 與傳統藍牙， 可手機遙控、 藍牙通訊。

PWM 控制

可控制 LED 亮度、 馬達速度、 Servo 角度。

ADC 類比輸入

讀取光敏電阻、 搖桿、 各種感測器。

Deep Sleep 省電模式

超低功耗， 適合電池供電 IoT 裝置。

Dual Core 雙核心

同時執行多工作業， 提升效能。

SPI / I2C / UART

支援多種通訊協定， 可連接 OLED、 RFID、LCD 等模組。

Touch Sensor

ESP32 支援觸碰感測， 可製作觸控按鈕。
